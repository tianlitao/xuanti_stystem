根据实际需求，完成毕业设计课题管理与选题的管理。能够实现院系信息、专业信息、学生信息、导师信息的维护功能，学生选题结果、提交毕业设计与论文以及指导老师对学生论文的指导与信息反馈功能等。
###
1.用户模块：区别指导教师与毕业生的登录。
###
2.新用户注册：指导教师与毕业生都要设定自己的用户名和密码，并填入自己的真实姓名和学号，登录后方可浏览相关信息。
###
3.选题模块：学生可以浏览毕业题目，并进行选择，且不能与其他同学重复。
###
4.毕业设计和论文的上传、查询与下载模块：学生可上传自己的毕业设计与论文，以便指导教师及时查看与指导。同时，指导教师也可将指导修改后的论文进行上传给学生。
